# 目标天体感知 — 不同类型天体的处理策略

本文档详述不同深空天体类型的差异化处理策略。

> **重要更新**：以下规则已自动落实到 `scripts/pipeline.py` 的 `apply_target_aware_safety_rules()` 函数中。当使用 `--target-type` 和 `--target-name` 参数时，管线会自动执行对应的安全规则（禁用危险步骤、修正参数）。AI 仍需了解这些规则的原理，但不需要手动禁用步骤或调整参数。

---

## 0. 识别流程

AI 识别天体类型的方式：

1. **用户告知** — 用户说「处理这张M42的照片」，直接获取
2. **FITS header** — 读取 FITS 文件中的 OBJECT 字段（`--target-name` 未指定时自动读取）
3. **AI 视觉识别** — AI 读取图像后判断天体类型
4. **询问用户** — 以上都无法确定时，主动询问用户

识别后通过 `--target-type <type> --target-name <name>` 传入管线，安全规则将自动生效。

## 1. 发射星云 (Emission Nebulae)

**代表**: M42 猎户座大星云、NGC 7000 北美星云、M16 鹰状星云、M8 礁湖星云、NGC 2237 玫瑰星云

**物理特征**: 高温恒星电离周围氢气体 → Hα (656nm) 红色主导 + OIII (500nm) 青蓝辅色

### 处理策略

| 参数 | 调整方向 | 原因 |
|------|---------|------|
| **拉伸** | 偏保守 (factor ×0.8) | 发射星云通常已有足够亮度，过强拉伸会导致核心过曝成白色 |
| **HDR** | 偏强 (+0.1) | Hα核心极其明亮，HDR压缩才能让核心和外围结构同时可见 |
| **降噪** | 标准 | 星云本身的"烟雾感"不是噪声，不要过度降噪抹掉 |
| **色彩** | Hα红不偏品红，OIII青不偏电蓝 | 发射星云的颜色是最容易被过度饱和毁掉的 |
| **星点** | 正常缩星 | M42核心的四边形星团不要缩掉 |
| **锐化** | 偏保守 (×0.7) | 星云纹理本身是"柔和的"，过锐会制造不存在的硬边 |

### 特别关注

- **M42 核心问题**: Trapezium 四合星区域极易过曝。考虑两段式拉伸：核心区域单独 mask 保护
- **NGC 7000**: 极其弥散的 Hα 外围，拉伸不足会丢失"墨西哥湾"海岸线
- **M16**: 创生之柱的暗尘埃边缘要锋利，锐化重点在暗尘埃边界而非星云内部
- **NGC 6888 眉月星云**: Wolf-Rayet 风泡壳层是主体。宽场 RGB/LP 数据中目标可能只占画面很小区域，但默认应保留宽场构图，只裁无效黑边；自动定位用于局部增强蒙版，不用于放大裁切。增强重点是外壳弧线和内部丝状结构。背景大面积红色可能是天鹅座真实 Hα 云气，不应直接用 DBE 或灰度世界校正抹掉。

### RGB/OSC 发射星云专用流程

当输入是彩色相机 RGB、双窄带已合成 RGB 或 LP 滤镜集成图时：

1. 诊断为无明显梯度时跳过 DBE。
2. 使用 `color_mode=emission`：只减每通道暗部基线，并仅在亮星软蒙版内校正星色。
3. 使用共享亮度标尺的 emission stretch，避免独立通道归一化抹平 Hα/OIII 比例。
4. 局部增强只作用于目标区域的亮度细节。
5. 低噪声长曝光数据默认不做最终降噪；视觉确认颗粒明显时再轻量加入。

## 2. 反射星云 (Reflection Nebulae)

**代表**: M45 昴星团、NGC 7023 鸢尾花星云、IC 2118 女巫头星云

**物理特征**: 尘埃反射附近恒星的光 → 蓝色调主导。本身不发光，所以非常暗弱。

### 处理策略

| 参数 | 调整方向 | 原因 |
|------|---------|------|
| **拉伸** | 激进 (+20%) | 反射星云极暗，需要更强的拉伸才能揭示 |
| **HDR** | 偏弱 (-0.1) | 反射星云没有发射星云那种极端核心-外围对比 |
| **降噪** | 偏强 (+15%) | 暗弱信号伴随高噪声，但注意不要抹掉微妙的反射渐变 |
| **色彩** | 保护蓝调，不要饱和到"电蓝" | 反射星云的美在于柔和的蓝，过度饱和会失真 |
| **星点** | 昴星团不缩星！其他可适度缩星 | M45 的七姐妹星是主角，缩星会毁掉整个画面 |
| **锐化** | 标准 | 尘埃纹理需要适度锐化来揭示 |

### 特别关注

- **M45 特殊处理**: 不要缩星、不要分离星点。昴星团的核心魅力就是包裹在蓝色反射星云中的明亮星团。缩星或分星会彻底毁掉这张图
- **IC 2118**: 极其暗弱，可能需要拉伸因子 100+

## 3. 星系 (Galaxies)

**代表**: M31 仙女座大星系、M51 漩涡星系、M101 风车星系、M33 三角座星系

**物理特征**: 核心是老年恒星（黄色），旋臂是年轻恒星（蓝色）+ HII 区（粉色小点）。尘埃带在旋臂内侧呈暗棕色。

### 处理策略

| 参数 | 调整方向 | 原因 |
|------|---------|------|
| **拉伸** | 标准 | 星系动态范围大，核心亮、外围暗 |
| **HDR** | 强 (+0.15) | 核心-旋臂的极端动态范围必须压缩才能同时看到两者 |
| **降噪** | 偏保守 | 银河外围极暗，降噪容易抹掉微弱的恒星流和外围结构 |
| **色彩** | 核心黄、旋臂蓝、HII粉 | 星系色彩应该"自然摄影"风格，不做发射星云式的激进饱和 |
| **星点** | 缩星重要！ | 前景银河恒星会干扰对星系结构的观察 |
| **锐化** | 偏强 | 尘埃带需要锐利边缘，旋臂纹理需要清晰 |

### 特别关注

- **M31 外围**: 如果数据足够深，不要过度拉伸外围——微弱的恒星流是真实结构，不要让降噪把它们当噪声抹掉
- **M51**: 潮汐桥是核心特征，确保拉伸后可见
- **前景星**: 分离和缩星时注意区分前景银河恒星和星系内部的星点

## 4. 球状星团 (Globular Clusters)

**代表**: M13 武仙座球状星团、M3、M5、M22

**物理特征**: 古老恒星密集球体。核心恒星极密，向外逐渐稀疏。

### 处理策略

| 参数 | 调整方向 | 原因 |
|------|---------|------|
| **拉伸** | 偏保守 | 核心恒星已经够亮 |
| **HDR** | 强 | 核心密集恒星和外围稀疏恒星需要同时清晰 |
| **降噪** | 极保守 | 外围暗弱恒星容易被误判为噪声 |
| **色彩** | 微调 | 球状星团色彩自然：蓝水平分支星 + 红巨星 |
| **星点** | **绝对不要分离和缩星！** | 星点就是目标本身，分离或缩星会毁灭图像 |
| **锐化** | 标准 | 帮助解析紧密恒星 |

## 5. 行星状星云 (Planetary Nebulae)

**代表**: M57 环状星云、M27 哑铃星云、NGC 2392 爱斯基摩星云

**物理特征**: 死星抛射的外壳。小、亮、结构复杂。OIII 强烈。

### 处理策略

| 参数 | 调整方向 | 原因 |
|------|---------|------|
| **拉伸** | 标准 | 星云本身很亮 |
| **HDR** | 强 | 内外壳结构需要极端HDR才能同时揭示 |
| **降噪** | 偏强 | 目标区域小，背景噪声可以大胆处理 |
| **色彩** | OIII青蓝要精致 | 行星状星云 OIII 极强，但不要变成"荧光蓝" |
| **星点** | 缩星 | 中心白矮星保留，周围恒星缩掉以突出星云结构 |
| **锐化** | 偏强 | 小尺度结构需要锐化才能看清 |

---

## 6. 暗星云 / 分子云 (Dark Nebulae / Molecular Clouds)

**代表**: 马头星云 B33、蛇夫座 ρ 星云复合体

**物理特征**: 尘埃遮挡背景星光。核心特征不是"亮"，而是"暗"。

### 处理策略

| 参数 | 调整方向 | 原因 |
|------|---------|------|
| **拉伸** | 关键：拉伸背景电离气体，让暗星云以"剪影"呈现 |
| **色彩** | 背景 Hα 红与尘埃暗棕形成对比 |
| **星点** | 标准缩星 | 星点干扰暗星云轮廓 |

---

## 7. 宽场星野 (Wide Field / Milky Way)

**代表**: 银河中心、星座广域

**处理策略**: 先完成镜头平场、渐晕和地平光害诊断，再决定 DBE。极多星点时可适度缩星，但必须保护星座主星和银河尘埃。地面前景与天空层分别配准处理，全景各面板先做低频匹配再统一拉伸。详细流程见 `special_targets.md`。

---

## 8. 超新星残骸 (Supernova Remnants)

**代表**: M1 蟹状星云、面纱星云、Simeis 147

**物理特征**: 激波形成的极暗丝状壳层，Hα、OIII 和 SII 空间分布差异明显。

### 处理策略

| 参数 | 调整方向 | 原因 |
|------|---------|------|
| **拉伸** | 无星层偏强、星点层保守 | 暗弱壳层容易被密集星场淹没 |
| **HDR** | 按目标调整 | M1 需要保护亮核，宽场面纱通常不需强 HDR |
| **降噪** | 线性阶段保守执行 | 极暗信号拉伸后噪声会快速放大 |
| **色彩** | 依据真实 Hα/OIII/SII 信号 | 弱 OIII 不可强行归一化成电蓝 |
| **星点** | 可分离或缩星，但保护细丝 | 去星算法容易把冲击前沿误判为星芒 |
| **锐化** | 多尺度蒙版局部增强 | 全图锐化会制造断丝和噪声边缘 |

背景模型中若出现完整弧线或丝状结构，必须回退 DBE。详细流程见 `special_targets.md`。
