# Photoshop 天文后期处理工作流程

## 简介
Photoshop 是通用的图像编辑软件，适合天文图像的精细调整、合成和艺术化处理。

## 前期准备

### 文件导入
```
1. 从 Siril/PixInsight 导出 16-bit TIFF
2. Photoshop → 文件 → 打开
3. 推荐色彩空间：
   - 工作空间：Adobe RGB (1998) 或 ProPhoto RGB
   - 或保持嵌入的配置文件
```

### 图层管理
```
推荐图层结构（从下到上）：
- 背景图层（原始图像）
- 背景校正图层
- 星点处理图层
- 星云/目标增强图层
- 色彩调整图层
- 锐化和降噪图层
- 最终调整图层
```

## 核心处理流程

### 1. 基础调整（曲线和色阶）

#### 1.1 曲线调整（Curves）
```
图层 → 新建调整图层 → 曲线

常用调整：
- RGB 综合曲线：S 形曲线增强对比度
- 分别调整 R/G/B 通道：色彩平衡
- 保护暗部：在曲线底部创建控制点

技巧：
- 使用蒙版限制调整范围
- 降低图层不透明度（30%-70%）获得自然效果
```

#### 1.2 色阶调整（Levels）
```
图层 → 新建调整图层 → 色阶

设置：
- 输入色阶：移动黑点和白点
- 中间调滑块：调整整体亮度
- 输出色阶：扩展动态范围

通道分别调整：
- 红色通道：调整冷暖平衡
- 绿色通道：调整植被/星云色调
- 蓝色通道：调整天空深度
```

### 2. 背景处理

#### 2.1 背景均匀化
```
原则：
1. 优先在 Siril 或 PixInsight 的线性数据阶段完成背景建模
2. Photoshop 只适合对已经确认的低频残差做保守、可逆的调整
3. 使用调整图层和大尺度软蒙版，比较处理前后并保留原始背景
4. 不使用污点修复、仿制图章、内容识别或生成式填充来“修复”天区背景；
   这些工具可能删除或伪造真实星云、IFN、暗尘埃和微弱星系结构
```

#### 2.2 背景变暗/变亮
```
1. 新建曲线调整图层
2. 调整背景亮度
3. 使用蒙版：
   - 填充黑色（隐藏）
   - 用白色画笔涂抹背景区域
   - 羽化边缘（200-500px）
```

### 3. 星点处理

#### 3.1 星点缩小（Star Shrink）
```
使用 Astronomy Tools Action Set (需安装)

或手动方法：
1. 复制背景图层（Ctrl+J）
2. 滤镜 → 其他 → 最小值...
- 半径应相对当前输出尺寸和星点 FWHM 选择，从最小有效值开始
   - 保留：圆度
3. 降低不透明度到 30%-50%
4. 添加蒙版隐藏过暗区域
```

#### 3.2 星点锐化
```
1. 复制图层
2. 滤镜 → 锐化 → 智能锐化
- 数量和半径从低值开始，通过 100% 视图检查星点边缘
- 不对饱和星、衍射芒和背景噪声统一锐化
- 出现白边、黑圈或颗粒增强时降低强度或回退
3. 添加蒙版：
   - Ctrl+Alt+点击星点图层缩略图（载入星点选区）
   - 反选（Ctrl+Shift+I）
   - 填充黑色（隐藏非星点区域）
```

#### 3.3 星点着色
```
1. 新建曲线调整图层
2. 分别在 R/G/B 通道调整
   - 热星：增加红色和蓝色
   - 冷星：增加蓝色
3. 使用星点蒙版限制效果
```

### 4. 星云/目标增强

#### 4.1 结构增强（High Pass Filter）
```
1. 复制背景图层
2. 滤镜 → 其他 → 高反差保留
   - 半径：3-10 像素（根据目标大小）
3. 混合模式：柔光 或 叠加
4. 降低不透明度到 30%-70%
```

#### 4.2 使用 Camera Raw 滤镜
```
滤镜 → Camera Raw 滤镜 (Shift+Ctrl+A)

调整：
- 清晰度、去朦胧和纹理从低值逐步增加
- 使用目标蒙版，避免增强背景噪声、星点光晕和梯度残差
- 每次调整后检查弱结构是否在原始数据中真实存在
- 使用径向滤镜局部增强目标
```

#### 4.3 渐变映射（增强色彩）
```
1. 新建渐变映射调整图层
2. 选择天文预设（黑-蓝-红-白）
3. 混合模式：柔光 或 颜色
4. 降低不透明度到 10%-30%
```

### 5. 降噪和锐化

#### 5.1 智能降噪
```
方法 1：Camera Raw 降噪
滤镜 → Camera Raw 滤镜
- 明亮度和颜色降噪按 100% 视图中的实际噪声逐步调整
- 使用蒙版保护星点、细丝、尘埃边缘和微弱星系外围
- 背景出现塑料感、色块或细节消失时立即降低强度

方法 2：第三方插件
- Topaz DeNoise AI
- Neat Image
- Noiseware
```

#### 5.2 蒙版锐化（保护背景）
```
1. 创建星点和目标选区
   - 在通道面板创建新通道
   - 用计算命令（图像 → 计算）提取亮部
2. 载入选区（Ctrl+点击通道缩略图）
3. 选择 → 反选（背景）
4. 滤镜 → 锐化 → 智能锐化
   - 仅锐化非背景区域
```

### 6. 色彩调整

#### 6.1 色彩平衡
```
图像 → 调整 → 色彩平衡 (Ctrl+B)

调整：
- 不预设阴影必须偏蓝、高光必须偏暖
- 依据已校准的星色、背景和目标物理类型调整
- 窄带配色属于映射选择，应明确记录通道到颜色的关系
```

#### 6.2 色相/饱和度
```
图层 → 新建调整图层 → 色相/饱和度 (Ctrl+U)

调整：
- 主饱和度从低值开始，按星色和目标信号逐步调整
- 分别调整颜色通道：
  - 红色：增强发射星云
  - 蓝色：增强反射星云
  - 黄色：增强恒星
```

#### 6.3 可选颜色
```
图层 → 新建调整图层 → 可选颜色

调整原则：
- 不使用固定的可选颜色配方处理所有目标
- 先确认颜色偏差来自校准、梯度、通道裁切还是审美映射
- 使用蒙版限制调整，并检查恒星颜色和背景是否同时保持合理
```

### 7. 合成技术

#### 7.1 HDR 合成
```
1. 准备不同曝光的版本
   - 暗版本（保留亮部）
   - 亮版本（显示暗部）
2. 文件 → 自动 → HDR Pro
3. 调整 HDR 色调
4. 使用蒙版融合最佳部分
```

#### 7.2 多窄带合成（Narrowband Combination）
```
如果分别从 Ha/OIII/SII 数据导出：
1. 打开各通道图像
2. 窗口 → 排列 → 平铺
3. 复制各通道到同一文档的不同图层
4. 分别赋予色彩：
   - Ha → 红色通道
   - OIII → 蓝色和绿色通道
   - SII → 红色通道（或 SHO 调色板）
5. 使用"应用图像"混合通道
```

### 8. 使用 Astronomy Tools Action Set

#### 8.1 安装
```
1. 下载 Astronomy Tools Action Set
2. 在 Photoshop 中：
   - 窗口 → 动作（Alt+F9）
   - 点击右上角菜单 → 载入动作
   - 选择下载的 .atn 文件
```

#### 8.2 常用动作
```
- Make Stars Smaller: 星点缩小
- Make Stars Larger: 星点放大
- Enhance DSO and Reduce Stars: 增强深空目标并减少星点
- Create Star Mask: 创建星点蒙版
- Hubble Palette (SHO): SHO 调色板合成
```

## 蒙版技术

### 亮度蒙版（Luminosity Masks）
```
手动创建：
1. 通道面板 → Ctrl+点击 RGB 通道（载入亮度选区）
2. 选择 → 存储选区 → 命名 "Bright"
3. 选择 → 修改 → 羽化（10-50px）

或使用插件：
- Lumenzia (推荐)
- Raya Pro
- TK Luminosity Masks
```

### 色彩蒙版
```
1. 选择 → 色彩范围
2. 吸管工具选择目标颜色
3. 调整容差（20-40）
4. 存储选区
```

## 输出和保存

### 保存 PSD（保留图层）
```
文件 → 保存为
- 格式：Photoshop (*.PSD;*.PDD)
- 图层：保留所有图层
- 最大兼容性：勾选
```

### 导出最终图像
```
文件 → 导出 → 存储为 Web 所用格式 (Alt+Shift+Ctrl+S)

设置：
- JPEG: 高质量（10-12）用于分享
- TIFF: 无压缩或 LZW 压缩用于打印
- PNG: 24 位用于网络分享
```

### 打印准备
```
图像 → 模式 → CMYK 颜色（用于专业打印）
图像 → 调整 → 色域警告（查看超出色域的颜色）
```

## PS 与天文专用软件的衔接

### 从 PixInsight 导入
```
1. PixInsight 导出 16-bit TIFF
2. 色彩空间：Adobe RGB 或 ProPhoto RGB
3. Photoshop 中：编辑 → 指定配置文件（保持一致）
```

### 从 Siril 导入
```
1. Siril 导出 16-bit TIFF
2. 可能需要：编辑 → 指定配置文件 → sRGB
```

## 性能优化

### 内存管理
```
编辑 → 首选项 → 性能
- 内存使用：70%-85%
- 历史记录状态：20-50
- 缓存拼贴大小：1024K-4096K（根据图像大小）
```

### 智能对象
```
- 转换图层为智能对象（右键 → 转换为智能对象）
- 好处：无损缩放、可逆滤镜
- 适合：Camera Raw 滤镜、第三方插件
```

## 推荐快捷键

```
- Ctrl+J: 复制图层
- Ctrl+Shift+N: 新建图层
- Ctrl+Alt+Shift+N: 新建空白图层
- Ctrl+[: 向下移动图层
- Ctrl+]: 向上移动图层
- Ctrl+G: 图层编组
- Ctrl+Alt+G: 创建剪贴蒙版
- Ctrl+点击图层缩略图: 载入图层透明度选区
- Q: 快速蒙版模式
- X: 切换前景/背景色
- D: 恢复默认颜色（黑/白）
```

## 常见问题

### 星点过亮
```
1. 使用星点蒙版
2. 曲线调整降低星点亮度
3. 或使用 Astronomy Tools 的 "Make Stars Smaller"
```

### 背景不均匀
```
1. 返回线性母版，在 Siril/PixInsight 中重新检查背景模型
2. 若只剩轻微低频残差，在 Photoshop 中使用可逆调整图层和大尺度软蒙版
3. 不使用修复画笔、仿制图章、内容识别或渐变映射覆盖真实天区结构
```

### 色彩暗淡
```
1. 先检查色彩校准、色彩空间和通道是否被裁切
2. 使用自然饱和度或曲线从低值逐步调整
3. 使用目标/色彩蒙版限制范围，并保留恒星颜色作为真实性检查
```
